This program simulates a basic ecommerce application such as Amazon.

Basic commands (CUSTS, BOOKS, ORDERS, BOOKS, SHIPPED) are functional
Intermediate commands (ORDER, SHIP, CUSTORDERS, NEWCUST, CANCEL) are functional
ORDERBOOK and ORDERSHOES are functional
Sort actions are functional

To see list of customers/products/orders enter prods/custs/orders
To add customers, enter newcust
To check shipped orders, enter shipped
Other commands can be seen on ECommerceUserInterface.java
